widget

select

An interactive list that lets the user pick a single option. Renders as an arrow-navigable menu in the terminal.

Properties

field type required description
id string yes Unique step identifier.
type "select" yes Must be "select".
label string yes Question displayed above the list.
options string[] yes Array of option values. Can be plain strings or objects with label/value.
default string no Pre-selected option value.
when expr no Conditional expression for skipping this step.
target path no Output path for the selected value.
hints boolean no Show navigation hints in terminal footer.

Example

yaml 
- id: framework
  type: select
  label: "Select framework"
  target: config.framework
  options:
    - React
    - Vue
    - Svelte
    - Angular
# Options with labels
- id: env
  type: select
  label: "Deploy target"
  target: config.env
  options:
    - label: "Production"
      value: prod
    - label: "Staging"
      value: stg
    - label: "Development"
      value: dev
