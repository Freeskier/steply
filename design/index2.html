CORE FEATURES

Built for production.

Schema-Aware Runtime

Every widget backed by typed schema. Runtime validates configuration as you build—no surprises at execution time.

  • Type-safe configuration
  • Real-time validation
  • Auto-completion support
widgets:
  - type: text_input
    validators:
      - type: min_length
        value: 3

Conditional Logic

Gate steps and widgets with expressions. Build dynamic flows that adapt to user input and runtime context.

  • Expression-based conditions
  • Dynamic step gating
  • Context-aware flows
- id: advanced
  when:
    all:
      - ref: show_advanced
        equals: true

Tasks & Subscriptions

Execute shell commands, parse JSON, trigger on input changes. Wire up runtime tasks with debounce and timeout control.

  • Shell command execution
  • JSON/text parsing
  • Debounced subscriptions
tasks:
  - id: search
    kind: exec
    program: python3
    parse: json
